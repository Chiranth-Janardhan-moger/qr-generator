# Modern QR Code Generator

A sleek, lightweight, and modern QR Code generator built with vanilla HTML, CSS, and JavaScript. 

This application allows users to instantly convert any URL or text into a beautiful, scannable QR code. It features a clean Glassmorphism-inspired UI with dynamic background blobs and smooth micro-animations.

## 🚀 Features

- **Instant Generation**: Type any URL or text and generate a QR code instantly.
- **Download Support**: Easily download the generated QR code as a high-quality PNG image.
- **Modern UI/UX**: Premium aesthetic with a blurred glass container, animated background blobs, and sleek hover effects.
- **Responsive Design**: Works perfectly across all device sizes (mobile, tablet, desktop).
- **Zero Dependencies**: Pure HTML/CSS/JS frontend relying only on the lightweight `qrcode.js` library via CDN.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Custom properties, Flexbox, transitions, and animations.
- **JavaScript (ES6)**: DOM manipulation and QR code generation logic.
- **qrcode.js**: Client-side QR code generation.

## 📋 Getting Started

### Prerequisites

No build tools or package managers are required! All you need is a modern web browser.

### Installation & Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/Chiranth-Janardhan-moger/qr-generator.git
   ```
2. Navigate to the project directory:
   ```bash
   cd qr-generator
   ```
3. Open `index.html` in your favorite web browser.
4. Enter any text or URL into the input field and click **Generate QR Code**.
5. Click **Download PNG** to save the QR code to your device.
